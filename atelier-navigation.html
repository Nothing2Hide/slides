  1. Authenticité : certificats https = carte d'identité d'un site
  2. Confidentialité : le trafic vers le site est chiffré
  3. Intégrité : les données envoyées ne peuvent être modifiées

Solutions en carton

  • Utiliser IE Firefox ou Chrome
  • Découvrez les traces que votre navigateur laisse en ligne
  • Apprenez à les gérer : cookies tiers, Do Not Track, empreinte numérique
  • Avec des extensions utiles : https everywhere, Disconnect

Solutions en béton

  • Chiffrer votre connection avec un VPN
  • Utiliser le navigateur Tor contre les écoutes indiscrètes
Tor ou un VPN ned vous rend pas anonyme à tous les coups. Exemple : se connecter à son compte facebook nom.prenom avec un VPN ou Tor, c'est rompre son anonymat.
