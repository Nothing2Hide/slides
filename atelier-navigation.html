http S ?

  1. Authenticité : certificats https = carte d'identité d'un site
  2. Confidentialité : le trafic vers le site est chiffré
  3. Intégrité : les données envoyées ne peuvent être modifiées

Solutions #1

  • Utiliser IE Firefox ou Chrome
  • Utilisez des extensions utiles : https everywhere
  • Découvrez les traces que votre navigateur laisse quand même en ligne

Solutions #2

  • Chiffrer votre connection avec un VPN
  • Utiliser le navigateur Tor contre les écoutes indiscrètes
Tor ou un VPN ne vous rend pas anonyme à tous les coups. Exemple : se connecter à son compte facebook nom.prenom avec un VPN ou Tor, c'est rompre son anonymat.
