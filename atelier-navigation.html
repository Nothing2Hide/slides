  1. Authenticité : certificats https = carte d'identité d'un site

Solution

  1. Utiliser IE Firefox ou Chrome
  2. Installez Duck Duck Go Privacy Essentials sur Firefox ou Chrome
  3. Utilisez le navigateur Duck Duck Privacy Browser sur Android ou iOS
