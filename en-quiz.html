Question #1

zUo!7$r → Is this a robust password?

NO

The use of uppercase, lowercase letters, numbers and special characters does not garantee that a password is safe. The number of character in a password is the most important. So forget about passwords and use passphrases.

Question #2

Are my files protected is if lock my session on my computer using a password passphrase ?

NO

Only the use of encryption can guarantee your files are safe. VeraCrypt and PGP are great open source software for disk encryption.

Question #3

Which is the safest messaging app?

  1. Whatsapp
  2. Messenger
  3. Signal
  4. Jitsi
  5. Skype
  6. Telegram
  7. Wire

It depends.

All these apps offer end to end encryption. However only Signal, Wire and Jitsi have been fully audited.

Question #4

Is it safe to use a public Wifi network?

NO

When you visit a website or any online service, your data goes through many different computers (wifi router, your ISP server, Internet routers). Each of these computers can access to your unencrypted data. You should always secure your surf using https, a VPN or Tor.
